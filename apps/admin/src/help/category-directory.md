# Категории

Технический раздел показывает дерево нормализованных категорий StroyHub.
Он нужен для проверки taxonomy и перехода к quality review.

![Категории](/help/screenshots/category-directory.png)

## Блоки на скриншоте

1. **Техническое**: служебные разделы админки.
2. **Taxonomy**: дерево normalized categories.
3. **Переход к качеству**: открывает страницу unmatched category groups.
4. **Ветка категорий**: parent category, children, slug и счетчики товаров.

## Как использовать

Открывай этот раздел, когда нужно понять, есть ли уже подходящая категория для
товаров из review. Для массовых unmatched групп сначала смотри
**Категоризация**, затем возвращайся сюда, чтобы выбрать правильный target.

## Когда менять taxonomy

- Нет подходящей leaf category.
- Существующая категория слишком широкая для нормализации.
- Несколько источников стабильно используют разные raw categories для одной
  товарной области.

## Ограничение

Страница показывает дерево, но не должна становиться основным инструментом
массового редактирования. Массовые решения должны проходить через rules,
fixtures и review.
