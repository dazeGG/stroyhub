# Товары

Раздел **Товары** показывает исходные карточки товаров, которые уже лежат в базе.

![Каталог товаров](/help/screenshots/catalog.png)

## Что смотреть

- название товара и source product id;
- магазин и source id магазина;
- нормализованную категорию и исходную `category_raw`;
- последнюю цену;
- время последнего наблюдения.

## Фильтры

- поиск по названию;
- нормализованная категория;
- магазин;
- сортировка по свежести, цене, названию или магазину.

## Когда открывать issue

- много товаров одного типа попали в неправильную категорию;
- источник начал отдавать пустые или странные названия;
- цены выглядят в неверном масштабе;
- один и тот же source product постоянно дублируется.

## Полезные переходы

Ссылка **История цен** в строке товара открывает ценовые наблюдения по этой карточке.
