# Источники товаров

Раздел связывает бизнесовый магазин StroyHub с конкретными source shops из
2GIS, официальных API и HTML-каталогов.

![Источники товаров](/help/screenshots/shops.png)

## Блоки на скриншоте

1. **Источники**: навигация к магазинам, новым source candidates и загрузкам.
2. **Назначение экрана**: поясняет, что здесь редактируются только metadata,
   статусы и связи источников.
3. **Canonical магазины**: бизнесовые магазины, к которым можно привязывать
   несколько source shops.
4. **Source shops**: конкретные scrape targets с source id, типом источника,
   статусом, расписанием и связью с canonical shop.

## Что можно менять

- Display name, адрес, сайт и заметки canonical shop.
- Связь source shop с canonical shop.
- Статус и scrape metadata source shop.
- Large catalog mode для 2GIS источников, если он поддерживается.

## Когда идти в новые источники

Если source shop найден автоматически, но еще не подтвержден, он должен
появиться в **Новые источники**. Не создавай вручную дубль, пока кандидат не
проверен.

## Риски

Неправильная связь магазина смешивает офферы разных продавцов. Перед linking
сверяй название, адрес, сайт и source id.
