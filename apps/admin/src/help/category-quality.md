# Категоризация

Раздел помогает находить source products без normalized category и группировать
их по источнику, магазину и `category_raw`.

![Категоризация](/help/screenshots/category-quality.png)

## Блоки на скриншоте

1. **Каталог**: навигация к source products, canonical catalog и quality screen.
2. **Цель проверки**: описание задачи: найти шумные или неразобранные категории.
3. **Фильтры источника**: источник и магазин, чтобы сузить unmatched группы.
4. **Покрытие категориями**: всего товаров, процент покрытия, товары без
   категории и количество групп на review.
5. **Группа unmatched**: `category_raw`, магазин, источник и representative
   titles, по которым можно понять, куда мапить группу.
6. **Текст для issue**: готовый отчет, который можно скопировать в GitHub issue.

## Как работать

1. Начни с самого крупного источника.
2. Отфильтруй один магазин, если группы слишком разные.
3. Смотри representative titles, а не только `category_raw`.
4. Если mapping очевиден, заводи задачу на source category mapping.
5. Если товары разнородные, нужна новая rule или ручная проверка.

## Что не делать

Не добавляй новую категорию только по одному `category_raw`. Сначала нужно
понять реальные товары внутри группы.
