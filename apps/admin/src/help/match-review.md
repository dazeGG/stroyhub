# Сверка связей

Раздел показывает candidate pairs между source products. Это техническая
страница для оценки качества matching logic.

![Сверка связей](/help/screenshots/match-review.png)

## Блоки на скриншоте

1. **Техническое**: служебная навигация.
2. **Сверка связей**: заголовок и назначение страницы.
3. **Фильтры кандидатов**: источник, магазин, минимальная confidence и лимит.
4. **Пара для сравнения**: две карточки, confidence, метод и reason tokens.

## Как ревьюить кандидата

1. Сравни normalized titles.
2. Проверь категории и магазины.
3. Посмотри overlap tokens и left/right-only tokens.
4. Отмечай false positives, если отличаются размер, материал, сорт, упаковка
   или единица продажи.

## Важная граница

Эта страница помогает диагностировать matching logic. Основные operator actions
для принятия или отклонения кандидатов находятся в рабочих очередях и карточках,
где есть контекст качества.
