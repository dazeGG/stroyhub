# Новые источники

Раздел показывает найденные кандидаты магазинов из 2GIS. Оператор решает,
какие из них можно материализовать как scrape targets.

![Новые источники](/help/screenshots/shop-candidates.png)

## Блоки на скриншоте

1. **Очередь источников**: заголовок и краткое назначение страницы.
2. **Стратегии поиска**: группы discovery strategy, по которым были найдены
   кандидаты.
3. **Группа кандидатов**: найденные организации, их адреса, source id и статус.
4. **Проверка данных**: действие для live-проверки, что 2GIS source реально
   отдает каталог или полезные данные.

## Как принимать кандидата

1. Проверь название и адрес.
2. Сравни с уже существующими source shops.
3. Запусти проверку данных.
4. Материализуй только если кандидат не дублирует существующий магазин.

## Когда отклонять

- Это офис, склад, услуга или организация без каталога.
- Кандидат уже связан с существующим source shop.
- Источник не отдает товаров или отдает нерелевантные карточки.
