# Исходные товары

Раздел показывает source products: карточки как они пришли из 2GIS,
официальных API или HTML-парсеров. Это основной экран для проверки сырья.

![Исходные товары](/help/screenshots/source-products.png)

## Блоки на скриншоте

1. **Каталог**: навигация к source products, normalized catalog и categorization.
2. **Назначение списка**: поясняет, что здесь проверяются исходные карточки.
3. **Поиск и фильтры**: поиск по названию, категория, магазин и сортировка.
4. **Исходная карточка**: строка source product с source id, магазином,
   normalized category, `category_raw`, ценой и временем последнего наблюдения.
5. **Переход к ценам**: открывает карточку товара с price snapshots.

## Что смотреть

- Название и `normalized_title`.
- Магазин и `source_product_id`.
- `category_raw` и нормализованную категорию StroyHub.
- Последнюю цену и дату последнего наблюдения.

## Когда это проблема

- Товар выглядит не товаром: доставка, услуга, рекламная карточка.
- Цена явно в неверном масштабе.
- Категория не соответствует названию.
- Один и тот же source product дублируется из-за нестабильного source id.

## Следующий шаг

Для единичной карточки открывай **Карточку товара**. Для массовой проблемы по
категории переходи в **Категоризация**.
