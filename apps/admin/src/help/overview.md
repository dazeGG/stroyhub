# Помощь по админке

Админка StroyHub нужна для ежедневной проверки качества данных без терминала:
товары, цены, скрейпы, категории и кандидаты матчинга собраны в одном месте.

![Главный экран каталога](/help/screenshots/catalog.png)

## Как пользоваться

1. Запусти API и админку локально.
2. Открой нужный раздел в левом меню.
3. Используй фильтры сверху страницы, чтобы сузить данные до источника, магазина или статуса.
4. Если находишь повторяющуюся проблему, заводи GitHub issue с конкретными примерами.

## Локальный запуск

```bash
uv run uvicorn apps.admin_api.main:app --port 8001 --reload
```

```bash
cd apps/admin
pnpm dev
```

Админка открывается на `http://127.0.0.1:5173/` и ходит в API через локальный proxy.

## Что считать проблемой

- товар без цены, когда источник явно отдает цену;
- массово неверная категория;
- резкое падение `items_seen` или `items_saved` в скрейпах;
- повторяющиеся false positive кандидаты матчинга;
- источник начал отдавать другую структуру данных.
