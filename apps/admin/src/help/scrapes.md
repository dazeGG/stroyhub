# Скрейпы

Раздел **Скрейпы** показывает здоровье источников и магазинов.

![Статус скрейпов](/help/screenshots/scrapes.png)

## Что смотреть первым

- `Failed / partial`;
- магазины с просроченным `next_scrape_at`;
- последние запуски с ошибками;
- резкие изменения `seen` и `saved`.

## Фильтры

- источник;
- статус.

## Когда открывать issue

- один магазин регулярно падает;
- весь источник начал возвращать partial runs;
- `items_seen` резко меньше обычного;
- `next_scrape_at` не выставлен для активного магазина.

## Когда можно быстро поправить

- локально не был поднят Redis/PostgreSQL;
- неверно задано расписание или seed-метаданные одного магазина;
- ошибка явно связана с временной недоступностью внешнего источника.
