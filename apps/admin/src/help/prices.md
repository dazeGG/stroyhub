# История цен

Раздел **История цен** показывает все price snapshots для выбранной исходной карточки.

![История цен](/help/screenshots/prices.png)

## Что важно

StroyHub хранит наблюдения, а не только изменения цены. Если цена не менялась,
повторные строки все равно должны быть видны: это значит, что товар снова был найден при скрейпе.

## Как проверять

1. Выбери товар из списка или перейди из каталога.
2. Проверь порядок snapshot-строк по `parsed_at`.
3. Смотри на счетчики наблюдений, повторов цены и строк без цены.
4. Отдельно разбирай `Цена отсутствует`: это может быть parser gap или особенность источника.

## Когда открывать issue

- у активного товара перестали появляться новые snapshots;
- цена стала `null` для группы похожих товаров;
- цена парсится в неверной единице или масштабе.
