# Категории

Раздел **Категории** помогает разбирать товары без нормализованной категории.

![Качество категорий](/help/screenshots/categories.png)

## Что показывает экран

- всего товаров в выборке;
- процент покрытия категориями;
- количество товаров без категории;
- группы unmatched товаров по источнику, магазину и `category_raw`;
- representative titles для каждой группы.

## Как использовать

1. Начни с фильтра по источнику.
2. Смотри самые крупные группы.
3. Проверь representative titles.
4. Нажми **Скопировать отчет** и вставь текст в issue или PR.

## Когда открывать issue

- нужна новая ветка taxonomy;
- raw category слишком общая и требует продуктовых примеров;
- несколько источников называют одну область по-разному;
- нужен manual override workflow, а не простое правило.

## Когда можно быстро поправить

- source category alias очевидно мапится на существующую категорию;
- небольшая keyword rule закрывает группу без риска для существующих тестов.
